PARS Upgrade Project - WT

Working to update Course Management System from a PHP web framework to JavaScript Framework.

From the time I started this position, I've:

  • Worked to create a detailed UML to plan and explain the program
  • Created the first prototype using:
    • a MySQL database
    • Node.js
    • Express.js
    • Fetch API
    • JavaScript
    • CSS
    • HTML
  • Created the second prototype with a React Framework for the front end and connected it to the back-end of the first prototype
  • Continued making updates to improve efficiency and better meet the needs of users

I built this project on my own, but users are able to:

  • Create assessment plans for each section each year
  • Manage FCARs for each course
  • View reports to track progress towards meeting student outcome goals
  • Review assessment plans to see how well course learning objectives are met overall
  • Admins can manage years, courses, sections, users, student outcome goals, course learning objectives, and measurement methods
  • All users can manage their own account

Additional functionalities of the application:

  • For security purposes, users are signed in for only 6 hours
  • All components have error and loading states for better performance
  • Dropdowns are searchable to easily find content when dealing with many options
  • Strong responsive design setup for various screen sizes

This project has been a great learning experience and is the largest application I've built on my own at this point in my career.
