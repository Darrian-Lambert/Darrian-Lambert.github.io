Maps - Ogallala Commons

In the summer of 2023, I embedded two maps into websites for Ogallala Commons.

The markers themselves were designed in Illustrator. I made several concept designs and my mentors felt that the designs made to look like physical pins were the best choice. Here is the first concept of that marker:

I designed the application in VSCode using Google Maps API to give users an up to date dynamic map. The first map was embedded into ogallalacommons.org. It shows all internships managed by OC since 2007 as well as what each intern did. It is used as a complementary tool to show potential interns and partners the volume of work OC has done over the years through the internship program. The other map was embedded into myfoodshed.org. It is another searchable map that shows producers and what they produce. It is used to help users find producers who produce the product they are looking for.


To update these maps after I’m gone, I created a map updater to help OC update the map without needing to write the JavaScript. It’s a simple program that runs on the user’s browser. Users input the changes they want to make then input the generated list into the website. Since this application was only for internal usage, there are no publically viewable versions of the application for security purposes.
