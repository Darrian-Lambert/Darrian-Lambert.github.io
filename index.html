Work Experiencee

Ogallala Commons

Web Developer Intern

June 2023 - August 2023

In the summer of 2023, I did an internship for Ogallala Commons. For them, I embedded two maps onto their websites. The first one is a map that displays all the interns OC has managed since 2007. It works to show interns, family of interns, and organizations how OC has grown over the years. It can be accessed at https://ogallalacommons.org/intern-map/. The other map I made shows producers that work with Ogallala Commons via Foodshed. This map can help people find specific products among the various producers that work with this program. It can be accessed at https://myfoodshed.org/producer-map/. To help update these maps without needing to know JavaScript, I also made a simple browser application that OC administrative staff can use. Throughout this internship, I worked with HTML, CSS, and JavaScript on VSCode. I worked with GoogleMapsAPI so that users can have an accurate map to work with. The websites I embedded the maps on worked with WordPress.
